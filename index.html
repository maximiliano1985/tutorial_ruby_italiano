Benvenuto!

Questa repository GitHub raccoglie una serie di esercizi introduttivi a Ruby, organizzati per argomenti. La teoria alla base della programmazione in Ruby non è trattata in questa pagina, ma è possibile trovare validi riferimenti a questa pagina web. Se il link non funziona copia e incolla il seguente link sul tuo browser: http://www.ing.unitn.it/~bertolaz/2-teaching/2013-2014/AA-2013-2014-INF/learn.php

Scaricare la repository

Se avete installato gits sul vostro sistema potete scaricare questa repository digitando in un terminale:

$ git clone git@github.com:maximiliano1985/tutorial_ruby_italiano.git

Argomenti trattati

  1. Introduzione a Ruby, il condizionale if, e i cicli for e while.
  2. Condizionali multipli e cicli annidati.
  3. Introduzione agli array e le stringhe. Esercizi con gli array e le stringhe.
  4. Introduzione ai simboli e alle hash, con esercizi.
  5. Lettura e scrittura dei file di testo.

Nella cartella ulteriori esercizi si propongono esempi di programmazione di diversa difficoltà, ad esempio:

  • tre implementazioni, crescenti in difficoltà, del plot ascii di funzioni
  • il gioco della vita di Conway
