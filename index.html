Benvenuto!

Questa repository GitHub raccoglie una serie di esercizi introduttivi a Ruby, organizzati per argomenti. La teoria alla base della programmazione in Ruby non è trattata in questa repository, ma è possibile trovare validi riferimenti alla pagina web:

http://www.ing.unitn.it/~bertolaz/2-teaching/2013-2014/AA-2013-2014-INF/learn.php

Scaricare la repository

Se avete installato gits sul vostro sistema potete scaricare questa repository digitando in un terminale:

$ git clone git@github.com:maximiliano1985/tutorial_ruby_italiano.git
